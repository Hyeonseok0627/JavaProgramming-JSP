

라이브러리 다운로드 방법

구글에서 mvn 검색 -> 첫번째 사이트 접속 -> 검색창에 jstl 검색창

-> jstl 클릭 -> 버전 선택해서 클릭 -> files에서 view all 클릭 -> jar 확장자 파일 클릭하여 다운로드

여기에 주소가 있는데 이게 cdn라서 나중에 링크에 넣어서 연결할 때 이거 복붙해도됨


다운로드 받은 라이브러리 연동시키는 법

해당 프로젝트 -> src -> main -> webapp -> WEB-INF -> lib 여기에 복붙해서 넣어줌

추가한 라이브러리가 제대로 들어갔는지 확인하려면

Java Resources -> Libraries -> Web App Libraries 인에 해당 라이브러리가 있으면 정상적으로 넣은 것
